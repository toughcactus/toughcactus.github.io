Overview #

The Org Manual

样式 #

  1. 删除线 
    +Content+
  2. 下划线
    ox-hugo转markdown存在问题 
    _Content_
  3. 文本块 
    =test=
~test~
  4. 斜体 
    /test/
  5. 加粗 
    *test*

选项设置 #

  1. 关闭toc
    Table of Contents 

    #+OPTIONS: toc:nil

  2. 指定toc显示层级 

    #+TOC: headlines 2

  3. 上标和下标 

    a_b a^b

    关闭 

    #+OPTIONS: ^:nil

    需要使用{}转义 

    #+OPTIONS ^:{}

    a_{b} a^{b}

  4. 自动为标题添加标号
    关闭 

    #+OPTIONS num:nil

  5. 选项合并 

    #+OPTIONS: toc:nil  ^:{}
#+STARTUP: showall hidestars

  6. 显示所有内容 

    #+STARTUP: showall

  7. 只显示一级标题 

    #+STARTUP: overview

  8. 隐藏星号 

    #+STARTUP: hidestars

模板 #

Structure Templates
C-c C-, org-insert-structure-template

  1. 代码框
    填写编程语言 

    // C++样式

    #+begin_src LANGUAGE
#+end_src

  2. 文本框 

    文本框样式

    #+begin_example
#+end_example

  3. quote

    quote样式 

    #+begin_quote
#+end_quote
