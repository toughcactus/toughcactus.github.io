    Part 4 以Pawn为原点建立相对坐标系，修改平移的方向向量 APawn::AddMovementInput #

    控制Pawn前后左右移动，前进或后退传入ForwardVector和步长，左右移动传入RightVector和步长
    控制Character前后左右移动亦通过该接口



    参数 含义
    WorldDirection 移动角色的方向
    ScaleValue 方向正负
    bForce 使用默认值 
    AddMovementInput(GetActorForwardVector(), Amount);
// AddMovementInput(GetActorRightVector(), Amount);

    定义
    对UPawnMovementComponent::AddInputVector进行封装，修改方向向量


    Part 5 绕指定轴旋转PlayerController #

    旋转的控制权不在Pawn，在PlayerController。对PlayerController的旋转进行了封装。

    X轴向前，Z轴向上，Y轴向右。

    1. Roll
      绕X轴旋转，顺时针为正方向 YOZ平面
    2. Pitch
      绕Y轴旋转，向上为正方向 XOZ平面
    3. Yaw
      绕Z轴旋转，向右为正方向 XOY平面

    绕Y轴和Z轴旋转时，若无其他处理，可以直接将AddControllerPitchInput和AddControllerYawInput注册为旋转回调函数

    1. 绕Y轴旋转Controller APawn::AddControllerPitchInput #

    摄像机向上或向下运动，Pitch增加时向上旋转

    只旋转Camera，且还需在虚幻编辑器设置Camera组件



    2. 绕Z轴旋转Controller APawn::AddControllerYawInput #

    摄像机向左右移动，Yaw增加时，向右旋转

    同时旋转Camera和Mesh



    3. 绕X轴旋转Controller APawn::AddControllerRollInput #

    Roll增加时，顺时针旋转
