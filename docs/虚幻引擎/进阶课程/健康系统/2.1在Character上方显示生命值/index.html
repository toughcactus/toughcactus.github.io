    Part 3 为 Character 添加健康组件 #

    C++ STBaseCharacter

    1. 包含健康组件头文件
      头文件 前向声明 
      class USTHealthComponent;
      源文件 
      #include "Components/STHealthComponent.h"
    2. 添加健康组件
      数据成员 protected 
      UPROPERTY(VisibleAnywhere, BlueprintReadWrite, Category = "Health")
USTHealthComponent *Health;
    3. 初始化组件
      函数成员 private 在构造函数中调用
      无表示，不用设置上级组件 
      void ASTBaseCharacter::InitHealthComponent()
{
    Health = CreateDefaultSubobject<USTHealthComponent>("HealthComponent");
}

    Part 4 为 Character 添加文本组件 #

    C++ STBaseCharacter 使用UTextRenderComponent

    1. 包含头文件
      头文件 前向声明 
      class UTextRenderComponent;
      源文件 
      #include "Components/TextRenderComponent.h"
    2. 添加TextRender组件
      数据成员 protected 
      UPROPERTY(VisibleAnywhere, BlueprintReadWrite, Category = "Health")
UTextRenderComponent *HealthText;
    3. 初始化组件
      函数成员 private 在构造函数中调用
      有视觉表示，挂到根组件下面 
      void ASTBaseCharacter::InitHealthTextComponent()
{
    HealthText = CreateDefaultSubobject<UTextRenderComponent>("HealthTextComponent");
    HealthText->SetupAttachment(GetRootComponent());
}
    4. 每帧更新显示文本
      函数成员 private 在Tick函数中调用 
      void ASTBaseCharacter::DisplayHealthText()
{
    HealthText->SetText(FText::FromString(FString::Printf(TEXT("%.0f"), Health->GetHealth())));
}
    5. 检查组件是否成功初始化
      函数成员 private 在BeginPlay中调用
      使用IDE编译时，若运行游戏崩溃，可在IDE中查看断点 
      void ASTBaseCharacter::CheckHealth()
{
    check(Health);
    check(HealthText);
}
