    Part 3 在 HealthComponent 输出伤害来源信息 #

    C++ STHealthComponent
    在处理函数中，根据 DamageType 的具体类型，做出相应处理

    1. 包含头文件 
      #include "Dev/STFireDamageType.h"
#include "Dev/STIceDamageType.h"
    2. 输出伤害来源信息
      函数成员 private 在OnTakeAnyDamage中调用 
      void USTHealthComponent::HandleDamage(const UDamageType *DamageType)
{
    if (DamageType)
    {
        if (DamageType->IsA<USTFireDamageType>())
        {
            UE_LOG(LogHealthComponent, Display, TEXT("So Hooooooot !!!"));
        }
        else if (DamageType->IsA<USTIceDamageType>())
        {
            UE_LOG(LogHealthComponent, Display, TEXT("So Cooooooooold !!!"));
        }
    }
}
