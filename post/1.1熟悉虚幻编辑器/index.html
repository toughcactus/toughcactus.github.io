虚幻引擎 > 初级课程 > 虚幻编辑器 > 1.1熟悉虚幻编辑器

虚幻编辑器是统称，而不是一个具体的编辑器。其包含关卡编辑器，蓝图编辑器，材质编辑器等。

关卡编辑器是这些编辑器的中心，可以在内容浏览器选中关卡，或蓝图类，或C++类，或材质，或其他资产。

当我们打开虚幻编辑器时，打开的是关卡编辑器。

一、关卡编辑器布局

二、正中间 视口 世界场景

  1. 显示设置

    • 视图设置 设置视口的观察方式

    • 光照设置 设置光照渲染

    • 顶视图 + 线框 用来判断对齐与否

    • 显示FPS

      • 每秒传输帧数 Frame Per Second

      • 菜单

  2. Gizmo

    • 具有Gizmo的Actor对象可平移，缩放，旋转
    • 选择当前的操作，也可使用快捷键切换

三、下方 内容浏览器

  • 展示各种资产，如关卡，C++类，蓝图类，材质，声音资产，角色模型等

四、下方 输出日志

五、右上 世界大纲

  • 世界场景(World)中所有物体列表

六、右下 世界场景设置(World Settings)

  • 配置关卡

七、右下 细节面板

  • 在世界大纲选中物体，在细节面板对其进行设置

八、左侧 放置Actor

  • 列出常用Actor

九、工具栏

  1. 运行 或 停止 程序

  2. 在独立窗口运行

十、菜单栏 常用操作

  1. 打开 或 关闭 窗口

    • Window项

  2. 打开 虚幻编辑器偏好设置

    • Edit项(Windows系统)

  3. 打开 项目设置

    • Edit项

十一、内容浏览器 常用操作

  1. 创建资产

    • 在内容浏览器右键，或者点击 Add

  1. 在场景中应用资产

    • 把资产拖到视口中的物体

  2. 调整图标大小及其他设置

十二、关卡设置

  1. 创建Map

    • 新建文件夹Content/Map，保存到该目录下

  2. 设置关卡编辑器和游戏的默认载入关卡

    • 项目设置 > Project > Maps & Modes

      • 设置 Editor Startup Map
      • 设置 Game Default Map

  3. 编辑器默认载入OpenWorld关卡，在场景中做的所有操作若不保存则会丢失

十三、编辑器偏好设置

  1. General

    • 快捷键

      • 查询 或 自定义 快捷键
    • 外观
    • 源码编辑器

  2. 自定义快捷键

    • 编辑器偏好设置 > General > Keyboard Shortcuts > Main Frame
    • Ctrl + Shift + N

十四、项目设置

  1. 生成项目ID

    • Project > Description

  2. 设置默认Map

    • 见关卡设置

十五、快捷键

  1. 移动视口中的摄像机

    • 按住鼠标 左键 或 右键
    • ADWS对应左右前后，QE对应上下

  2. 切换物体变换模式

    • W，E，R对应位移，旋转，缩放

  3. 隐藏 或 恢复 非游戏元素的显示

    • G 进入到游戏模式

  4. 使选中物体落到最近的平面(Mac不生效)

    • End

  5. 创建C++类(自定义添加快捷键)

    • Ctrl + Shift + N

  6. 定位Actor Focus

    • 在世界大纲选中实例
    • F

  7. 停止游戏

    • Esc

  8. 释放鼠标控制，游戏仍在运行

    • S-F1

十六、恢复属性的默认值

  • 设置属性值之后，会出现 恢复图标
  • 点击恢复默认值

十七、在命令行使用虚幻编辑器打开项目

  • Windows

      start "E:\Epic/熟悉虚幻编辑器 Games\UE_5.1\Engine\Binaries\Win64\UnrealEditor.exe" "Dirk\Dirk.uproject"

  • MacOS

      open "/Volumes/虚幻引擎/Epic/熟悉虚幻编辑器Games/UE_5.1/Engine/Binaries/Mac/UnrealEditor.app"
  open "/Volumes/虚幻引擎/Projects/5.1.1/TheBasics/TheBasics.uproject"

十八、从Visual Studio启动虚幻编辑器

  • 创建项目之后，我们通常在VS中进行编译运行，从而打开虚幻编辑器

  • 已经打开虚幻编辑器，使VS项目附加到进程
