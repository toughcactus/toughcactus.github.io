虚幻引擎 > 基础课程 > 1.6使用UnrealBuildTool

  • 使用UnrealBuildTool时，依赖很多第三方工具，下载引擎时，虚幻引擎有把这些工具打包，并在脚本里指定了路径
  • 如果是Windows系统，直接使用UBT问题不大
  • MacOS中，建议使用脚本

一、路径

  1. UBT路径

    • EpicGames/UE_5.1/Engine/Binaries/DotNET/UnrealBuildTool/

  2. 脚本路径

    • EpicGames/UE_5.1/Engine/Build/BatchFiles/Mac/

  3. uproject文件路径

    • 5.1.1/TheBasics/TheBasics.uproject

二、使用UBT生成项目文件

  • 在uproject文件同级目录生成工程文件
  • 在MacOS上直接使用UBT，需要安装dotnet 6.0 Runtime

  1. 创建默认源码编辑器的工程文件

      #!/bin/bash
  UBT="EpicGames/UE_5.1/Engine/Binaries/DotNET/UnrealBuildTool/UnrealBuildTool"
  Project="Projects/5.1.1/TheBasics/TheBasics.uproject"

  EngineCCDst="EpicGames/UE_5.1/compile_commands.json"
  EngineCCSrc="EpicGames/UE_5.1/.vscode/compileCommands_Default.json"

  ProjectCCDst="Projects/5.1.1/TheBasics/compile_commands.json"
  ProjectCCSrc="Projects/5.1.1/TheBasics/.vscode/compileCommands_TheBasics.json"

  # 生成默认Source Code Editor的工程文件
  $UBT -projectfiles -project="$Project" -game -rocket -progress

  # 生成VSCode工程文件
  $UBT -projectfiles -project="$Project" -game -rocket -progress -VSCode

  # 生成XCode工程文件
  $UBT -projectfiles -project="$Project" -game -rocket -progress -XCodeProjectFiles

  cp $EngineCCSrc $EngineCCDst
  cp $ProjectCCSrc $ProjectCCDst

  2. VS Code

    • 使用 -VSCode

  3. XCode

    • 使用 -XCodeProjectFiles

三、使用脚本 ShooterGame 

  #!/bin/bash
  Script="EpicGames/UE_5.1/Engine/Build/BatchFiles/Mac"
  GenerateProjectFilesScript=$Script"/GenerateProjectFiles.sh"
  BuildScript=$Script"/Build.sh"

  Project="Projects/Examples/ShooterGame/ShooterGame.uproject"

  # 生成VSCode工程文件       
  $GenerateProjectFilesScript -projectfiles -project="$Project" -game -rocket -progress -VSCode

  # 编译       
  $BuildScript ShooterGameEditor Mac Development "$Project" -waitmutex -NoHotReload

四、使用脚本 TheBasics 

  #!/bin/bash
  Script="EpicGames/UE_5.1/Engine/Build/BatchFiles/Mac"
  GenerateProjectFilesScript=$Script"/GenerateProjectFiles.sh"
  BuildScript=$Script"/Build.sh"

  Project="Projects/5.1.1/TheBasics/TheBasics.uproject"

  # 生成VSCode工程文件       
  $GenerateProjectFilesScript -projectfiles -project="$Project" -game -rocket -progress -VSCode

  EngineCCDst="EpicGames/UE_5.1/compile_commands.json"
  EngineCCSrc="EpicGames/UE_5.1/.vscode/compileCommands_Default.json"

  ProjectCCDst="Projects/5.1.1/TheBasics/compile_commands.json"
  ProjectCCSrc="Projects/5.1.1/TheBasics/.vscode/compileCommands_TheBasics.json"

  cp $EngineCCSrc $EngineCCDst
  cp $ProjectCCSrc $ProjectCCDst

  # 编译       
  $BuildScript TheBasicsEditor Mac Development "$Project" -waitmutex -NoHotReload

  # 在虚幻编辑器中打开项目       
  open $Project
