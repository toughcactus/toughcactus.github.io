虚幻引擎 > 进阶课程 > 虚幻编辑器 > 1.7动画蓝图

  • 动画随着Character运动速度变化
  • brush类型是最简单的可编辑几何体，我们可以修改其顶点位置，调整其他参数，然后将其转换成静态网格体
  • CharacterMovement组件负责Character运动，其各种属性均和Character运动有关

一、重新设置平台

  1. 从场景中移除Floor

  2. 添加Box

  3. 恢复默认位置，设置brush长宽高

二、Character最大速度

  1. 在蓝图编辑器中打开BP_STBaseCharacter

  2. 选择CharacterMovement组件 > 细节面板 > Character Movement: Walking > MaxWalkSpeed

    • 场景中Character的最大运动速度
    • 单位cm/s
    • 虚幻引擎里的步长使用cm

三、获取实时速度

  • BP_STBaseCharacter > EventGraph

  1. GetVelocity函数

    • 获得Actor各个方向上的速度，保存在FVector中

  2. FVector::VectorLength

    • 获得标量

  3. 在屏幕上输出实时速度

    • PrintString对应AddOnScreenDebugMessage
  4. 可以修改MaxWalkSpeed，当速度改为1600时，可以看到单位时间内移动的距离增加，但跑步动画的速度一直不变
  5. 移除打印

四、创建动画蓝图(Animation Blueprint)的两种方式

  • 创建文件夹 Content/Players/Animations
  • 动画蓝图和Skeleton Mesh有关

  • 命名

    • 参考

    • 动画蓝图命名 ABP_打头

方法一 创建Skeleton Mesh的动画蓝图

  • 选中ExternalContent/Characters/HeroTPP/HeroTPP > 右键 > Create > AnimBlueprint

方法二 在指定位置创建动画蓝图

  1. 去到Content/Player/Animations > 右键 > Animation > Animation Blueprint

  2. 选择HeroTPP_Skeleton，命名为ABP_BaseCharacter

五、在蓝图编辑器中查看动画蓝图ABP_BaseCharacter

  1. EventGraph

    • 处理各种动画事件

    • 像使用Tick一样使用EventBlueprintUpdateAnimation

  2. AnimGraph

    • 处理所有动画

    • 将动画作为OutputPose的输入

  3. AssetBrowser

    • 可以从 Window > AssetBrowser 打开
    • 罗列了SkeletonMesh的所有相关动画

  4. 可以将Run_Fwd作为OutputPose的输入

    • 选中Run_Fwd > Details > Settings > 使能 LoopAnimation

六、设置BP_STBaseCharacter的AnimationMode为UseAnimationBlueprint

  • 之前AnimationMode的设置是UseAnimationAsset，而后选择Run_Fwd
  • BP_STBaseCharacter > Details > Animation > AnimationMode，选择 UseAnimationBlueprint

  • Animation > AnimClass，设置为ABP_BaseCharacter

七、查看Run_Fwd

  1. 去到Content/ExternalContent/Animations/TTP_Animations
  2. 双击Run_Fwd

  3. 可在AssetBrowser中选择其他动画

八、创建混合空间(Blend Space)资产

  • 动画混合(Animation Blend)

    • 流畅地从一个动画切入到另一个动画

  • 混合空间能在两个骨骼动画之间创建过渡动画

    • 使用插值，动画姿势的中间值
  • 1维混合空间即Blend Space 1D
  • 混合空间也和Skeleton Mesh有关

  • 命名

    • 参考
    • 混合空间命名 BS_打头

    • 1D混合空间命名 BS_打头

  • 目标是，速度从0-600，动画从Idle过渡到Run_Fwd

1. 创建BS_Locomotion

  • 去到Content/Player/Animation，右键

  • 选择HeroTPP_Skeleton，命名为BS_Locomotion

2. 配置BS_Locomotion

  • 双击打开BS_Locomotion

  • 配置轴

    • Asset Details > Axis Settings > Horizontal Axis

    • 设置名字

      • Name，设置为Velocity

    • 设置轴最大值

      • Maximum Axis Value，设置为600

    • 过渡区间 Grid Divisions

      • 当前值为4
      • 和可以添加的动画个数有关

  • 配置动画

    • 起点 Idle

    • 终点 Run_Fwd

    • Windows，按下Control键；MacOS，按下Command键

    • 不用按下鼠标键位，拖动绿色X在起点和终点间移动，可以看到过渡动画

九、在ABP_BaseCharacter中使用BS_Locomotion

AnimGraph

  1. 在AnimGraph中，使用BS_Locomotion作为OutputPose的输入

  2. 添加变量

    • Float
    • Velocity

  3. 在AnimGraph添加变量Velocity，此处只能获取变量值，我们要的就是获取变量值

  4. 将变量Velocity作为BS_Locomotion的输入

EventGraph

  1. 在EventGraph添加变量Velocity，选择设置变量值

  2. 在EventBlueprintUpdateAnimation之后，执行SetVelocity

  3. 变量Velocity的输入来自Actor速度

    • 获得当前Pawn实例的引用 GetPawnOwner
