虚幻引擎 > API > 虚幻C++ > 1.日志类别

  • 每个模块可以有自己的日志类别，定位问题会比较方便
  • 虚幻C++提供了日志类别的枚举类，包含各模块的全局日志类别
  • 可以使用全局日志类别，也可以自定义静态日志类别

一、自定义日志类别 DEFINE_LOG_CATEGORY_STATIC

Engine/Source/Runtime/Core/Public/Logging/LogMacros.h

静态日志类别只能在定义所在源文件内使用。

  DEFINE_LOG_CATEGORY_STATIC(LogSimpleGeometry, All, All)
  1. 日志类别 CategoryName

  2. 打印级别 DefaultVerbosity

    • 输出日志时，给定日志级别小于打印级别时，不予打印
    • 日志级别数值越大，级别越低

  3. 编译级别 CompileTimeVerbosity

    • 输出日志时，给定日志级别小于编译级别时，该语句不予编译

建议将编译级别和打印级别均设为All

二、全局日志类别

使用DECLARE_LOG_CATEGORY_EXTERN定义

  1. LogTemp

    • 打印级别为Log

      • 日志级别不低于Log时打印

    • 编译级别为All

      • 无条件编译
      UE_LOG(LogTemp, Log, TEXT("Hello Unreal!"));
  UE_LOG(LogTemp, Display, TEXT("Hello Unreal!"));
  UE_LOG(LogTemp, Warning, TEXT("Hello Unreal!"));
  UE_LOG(LogTemp, Error, TEXT("Hello Unreal!"));

  2. 使用DECLARE_LOG_CATEGORY_EXTERN定义的日志类别，编译级别均为All

    • 无条件编译
