虚幻引擎 > API > 虚幻C++ > 14.UWorld类型

  • 关卡编辑器中，视口所展示的，即场景，又叫做世界场景
  • Engine/Source/Runtime/Engine/Classes/Engine/World.h

一、头文件 

  #include "Engine/World.h"

二、获取程序运行时长 UWorld::GetTimeSeconds 

  if (GetWorld())
  {
    float Time = GetWorld()->GetTimeSeconds();
  }

三、创建Actor对象并对其调用BeginPlay UWorld::SpawnActor

动态创建Actor对象，在调用完Actor对象的构造函数之后，调用Actor对象的BeginPlay函数

  • 返回AActor指针

  1. InClass

    • 对象类型

  2. Location

    • 使用默认值，原点

  3. Rotation

    • 使用默认值，不发生旋转
  TSubclassOf<ASimpleGeometry> GeometryClass;
  GetWorld()->SpawnActor(GeometryClass);

四、创建Actor对象并对其调用BeginPlay UWorld::SpawnActor<T>

  1. Class

    • 对象类型

  2. Transform

    • 变换信息

  3. SpawnParameters

    • 使用默认值
  TSubclassOf<ASimpleGeometry> GeometryClass;
  const FTransform FTransform = FTransform(FRotator::ZeroRotator, FVector(0.0f, 300.0f, 300.0f));
  ASimpleGeometry *Geometry = GetWorld()->SpawnActor<ASimpleGeometry>(GeometryClass, FTransform);

五、创建Actor对象 UWorld::SpawnActorDeferred<T> 

  TSubclassOf<ASimpleGeometry> GeometryClass;
  const FTransform Transform = FTransform(FRotator::ZeroRotator, FVector(0.0f, 300.0f, 700.0f));
  ASimpleGeometry* Geometry = GetWorld()->SpawnActorDeferred<ASimpleGeometry>(GeometryClass, Transform);

  1. Class

    • 对象类型

  2. Transform

    • 变换信息

  3. Owner

    • 使用默认值

  4. Instigator

    • 使用默认值

  5. CollisionHandlingOverride

    • 使用默认值
