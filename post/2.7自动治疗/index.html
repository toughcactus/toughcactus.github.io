虚幻引擎 > 进阶课程 > 虚幻C++ > 2.7自动治疗

  • 治疗机制

    • 受到伤害则延缓治疗
    • 不再受到伤害才会自动治疗
  • 使用定时器实现

  • 获取定时器管理对象有两种方式

    • UActorComponent::GetOwner > AActor::GetWorldTimerManager
    • UActorComponent::GetWorld > UWorld::GetTimerManager
  • HealthComponent逻辑

一、改进更新生命值

  • 受到伤害和自动治疗均会修改生命值，随之修改HealthText

  • 添加函数

    • private
    • 在USTHealthComponent::OnTakeAnyDamage计算伤害时，改为调用SetHealth
      void USTHealthComponent::SetHealth(float NewHealth)
  {
      Health = FMath::Clamp(NewHealth, 0.0f, MaxHealth);
      OnHealthChanged.Broadcast();
  }

二、添加治疗参数

  • 使能自动治疗 AutoHeal
  • 首次治疗延后 HealDelay
  • 周期恢复生命值 HealRate
  • 恢复量 HealModifier
  UPROPERTY(EditDefaultsOnly, Category = "Auto Heal")
  bool AutoHeal = true;

  UPROPERTY(EditDefaultsOnly, Category = "Auto Heal", meta = (EditCondition = "AutoHeal"))
  float HealRate = 1.0f;

  UPROPERTY(EditDefaultsOnly, Category = "Auto Heal", meta = (EditCondition = "AutoHeal"))
  float HealDelay = 3.0f;

  UPROPERTY(EditDefaultsOnly, Category = "Auto Heal", meta = (EditCondition = "AutoHeal"))
  float HealModifier = 5.0f;

三、添加定时器

  1. 添加数据成员

    • private
      FTimerHandle HealTimer;

  2. 在源文件包含头文件

      #include "Engine/World.h"
  #include "TimerManager.h"

  3. 添加函数

    • private
    • 进入USTHealthComponent::OnTakeAnyDamage中调用
    • 恢复到满生命值时调用
      void USTHealthComponent::StopHealTimer()
  {
      if (AutoHeal && GetWorld())
      {
          GetWorld()->GetTimerManager().ClearTimer(HealTimer);
      }
  }

  4. 添加函数

    • 定时器回调函数
    • private
      void USTHealthComponent::HealUpdate()
  {
      SetHealth(Health + HealModifier);
      if (FMath::IsNearlyEqual(Health, MaxHealth))
      {
          StopHealTimer();
      }
  }

  5. 添加函数

    • 初始化定时器
    • private
    • USTHealthComponent::OnTakeAnyDamage最后调用，受到伤害后，延后开始自动治疗
      void USTHealthComponent::InitHealTimer()
  {
      if (!IsDead() && AutoHeal && GetWorld())
      {
          GetWorld()->GetTimerManager().SetTimer(HealTimer, 
                                                  this,
                                                  &USTHealthComponent::HealUpdate,
                                                  HealRate, 
                                                  true, 
                                                  HealDelay);
      }
  }

四、重写IsDead 

  bool IsDead() const { return FMath::IsNearlyZero(Health); }
