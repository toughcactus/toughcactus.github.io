虚幻引擎 > API > 虚幻C++ > 2.字符串

  • 虚幻C++提供3种字符串结构体

    • FString
    • FName
    • FText

一、TCHAR

  • API

  • ANSICHAR 或 WIDECHAR，取决于平台

    • 可以把ANSICHAR理解为char，WIDECHAR当作wchar
  • 虚幻C++支持跨平台，当我们使用其提供的类型时，可以忽略平台差异(如编码和字体)
  • 格式化输出中的%s匹配TCHAR *

二、TEXT宏

包裹字符串，返回TCHAR *

三、FString结构体

  • API
  • 容器类型，元素类型是TCHAR
  • 格式化输出中的%s无法匹配FString
  • 输出FString时，需将其显式转换为TCHAR *

返回可输出字符串 FString::operator*

  • 返回TCHAR *

字符串连接 FString::operator+

整型转字符串 FString::FromInt

浮点数转字符串 FString::SanitizeFloat

构造函数 接受cstring

格式化输出 FString::Printf

四、FName结构体

  • 和FString相比，FName是一个轻量级的字符串，其功能较少，不包含字符串操作
  • 用于设置组件名，玩家名，材质参数，骨骼身体部分等
