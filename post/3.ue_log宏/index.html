虚幻引擎 > API > 虚幻C++ > 3.UE_LOG宏

输出日志时，给出所属的日志类别和该条日志的级别。

当日志的级别小于日志类别的打印级别时，忽略；小于日志类别的编译级别时，不编译该条语句。

  • 输出格式 CategoryName: Verbosity: TEXT
  • 当日志级别大于Log时，输出日志级别
  1. 日志类别 CategoryName

  2. 日志级别 Verbosity

    • 虚幻引擎定义了日志级别的枚举类
    • 常用的有Log，Display，Warning和Error
    • 在编辑器的输出日志窗口，不同级别的日志以不同颜色显示

  3. 文本格式 Format

    • 不接受cstring，要求TCHAR *
    • 使用TEXT宏包裹

    • Format中的%s匹配TCHAR *

      • 如果匹配cstring，编译不会报错，输出可能是乱码
