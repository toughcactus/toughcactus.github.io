三、在屏幕输出信息 UEngine::AddOnScreenDebugMessage

  1. 消息标识 Key

    • Key相同的消息只显示第一个
    • 使用-1则一定会显示
  2. 文本显示时长 TimeToDisplay

  3. 文本颜色 DisplayColor

    • 可以使用FColor预定义颜色，也可以使用自定义的FColor对象
  4. 文本内容 DebugMessage

  5. 在世界场景中的位置 bNewerOnTop

    • 默认在顶部输出信息，为false则在屏幕底部输出

  6. 设置文字大小 TextScale

      if (GEngine)
  {
      GEngine->AddOnScreenDebugMessage(-1, 3.0f, FColor::Orange, TEXT("Hello"));
      GEngine->AddOnScreenDebugMessage(-1, 5.0f, FColor::Green, TEXT("World!"));
  }
