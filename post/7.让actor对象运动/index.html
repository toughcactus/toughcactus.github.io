  • 三种运动类型

    • 静止 Static

    • Z方向上余弦运动 Sin

      • 修改Z坐标

    • 绕Z轴旋转 Rotate

      • 修改 Yaw

一、创建枚举类 EMoveType 

  UENUM(BlueprintType)
  enum class EMoveType : uint8
  {
      Static,
      Sin,
      Rotate
  };

在蓝图编辑器中可使用该类型

二、创建结构体 FMoveSetting 

  USTRUCT(BlueprintType)
  struct FMoveSetting
  {
      GENERATED_USTRUCT_BODY()

      UPROPERTY(VisibleDefaultOnly, Category = "MoveSetting")
      float Amplitude = 50.0f;

      UPROPERTY(VisibleDefaultOnly, Category = "MoveSetting")
      float Frequency = 2.0f;

      UPROPERTY(VisibleInstanceOnly, Category = "MoveSetting")
      EMoveType MoveType = EMoveType::Static;

      FTransform InitialTransform;
  };

在蓝图编辑器中可使用该类型

六、获得程序运行时长，用来计算相对变换

  • AActor::GetWorld
  • UWorld::GetTimeSeconds
  if (GetWorld())
  {
      float Time = GetWorld()->GetTimeSeconds();
  }

七、Z方向上余弦运动

  1. 从初始变换提取位置信息

    • FTransForm::GetLocation
      FVector InitialLocation = MoveSetting.InitialTransform.GetLocation();

  2. 获取Actor对象当前位置信息

    • AActor::GetActorLocation
      FVector CurrentLocation = GetActorLocation();

  3. 根据程序运行时长计算相对旋转

    • 步长 Amplitude
    • 振幅 Frequency
      CurrentLocation.Z = InitialLocation.Z + MoveSetting.Amplitude * FMath::Sin(MoveSetting.Frequency * Time);

  4. 设置新位置信息

    • AActor::SetActorLocation
      SetActorLocation(CurrentLocation);

八、绕Z轴旋转

  1. 从初始变换提取旋转分量

    • FTransForm::Rotator
      FRotator InitRotation = MoveSetting.InitialTransform.Rotator();

  2. 获取Actor对象当前旋转信息

    • AActor::GetActorRotation
      FRotator CurrentRotation = GetActorRotation();

  3. 根据程序运行时长计算相对旋转

    • 步长使用Amplitude
      CurrentRotation.Yaw = (InitRotation.Yaw + Time * MoveSetting.Amplitude);

  4. 设置新旋转信息

    • AActor::SetActorRotation
      SetActorRotation(CurrentRotation);
