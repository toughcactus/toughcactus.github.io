C++ > 类型 > auto和decltype

一、decltype | 类型指示符

  • 根据变量或表达式得到变量类型
  • 分两种情形讨论，变量语义，或表达式语义
  // 变量语义
  decltype(variable);

  // 表达式语义
  decltype(expr);
  decltype((variable));

1. 变量语义

  • 得到变量类型
  • 类成员也是变量
  int a = 5, &r = a;
  decltype(a) b; // 变量语义，int
  decltype(r) r2 = b; // 变量语义，int &

2. 表达式语义

  • 表达式的值类别是纯右值，得到非引用类型，变量类型为值类型

      #include <iostream>
  int fcn(int a) { return ++a; }
  int main()
  {
      decltype(fcn(3)) b = 6; // int
      return 0;
  }

  • 表达式的值类别是泛左值，得到引用类型，基本类型是对象的类型

    • 左值，得到左值引用类型

        int a = 5;
  decltype((a)) b = a; // 左值引用，基本类型为int

  double d = 3.14;
  double &rd = d;
  decltype((++rd)) rd2 = d; // 左值引用，基本类型为double

    • 速亡值，得到右值引用类型

        #include <iostream>
  int &&fcn(int a) { return ++a; }
  int main()
  {
      decltype(fcn(3)) b = 6; // 右值引用，基本类型为int
      return 0;
  }

二、auto | 类型说明符

  • 根据右侧表达式推断变量类型
  • 在一条语句中定义多个变量时，基本类型必须一致
  auto v = expr;

1. 得到的类型是明确对象的类型

  • 如果表达式类型是引用，auto得到其对应对象的类型
  int a = 5, &r = a;
  auto b = r; // b的类型是int

2. 如果对象有顶层const，舍弃 

  const int ci = 4;
  auto i = ci; // i的类型是int
  const auto ci2 = 3; // ci2的类型是const int

3. 如果对象具有底层const，保留 ｜ 指针 

  const int ci = 4, *p = &ci;
  auto p2 = p; // p2的类型为const int*

4.使用auto定义指针

  • 如果表达式类型是指针，将得到一个指针

    • 取地址运算符返回一个指针对象
      int a = 5, *p = &a;
  auto p2 = p; // p2是指针
  auto p3 = &a; // p3是指针

  • 使用类型修饰符*，要求右侧表达式类型是指针

      auto p3 = &a; // p3是指针
  auto *p4 = p2; // p4是指针

5. 使用auto定义左值引用

  • 使用类型修饰符&，要求右侧表达式是左值
  • 如果表达式的值类别是左值，使用类型修饰符&或&&都将得到一个左值引用
  int a = 5, &r = a;
  auto &r2 = a; // r2的类型是int &
  auto &&r3 = b; // r3的类型是int &

6. 使用auto定义右值引用

  • 使用类型修饰符&&，要求右侧表达式是右值

7. 使用类型修饰符*或&时，如果对象具有顶层const，变量具有底层const 

  const int ci = 4;
  auto &cr = ci; // cr的类型是const int&
  const auto &cr2 = ci; // cr2的类型是const int&
  auto p = &ci; // const int*
