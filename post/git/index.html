一、安装与配置

  1. 官网下载git-2.40.0 | 安装 make install | xcode自带

  2. 本地配置git账户

      git -v
  # 配置账户信息
  git config --global user.name "USERNAME"
  git config --global user.email "EMAIL-ADDRESS"

  3. 配置ssh

      # 生成秘钥 | 拷贝公钥到github
  ssh-keygen
  cat /Users/USERNAME/.ssh/id_rsa.pub

二、本地初始化 和Github项目同步 

  git init
  git remote add cute git@github.com:xxx/xx.git # cute为上游别名
  git remote -v
  git fetch cute # 拉取上游内容

三、创建分支与切换 

  git branch -a # 查看当前分支
  git checkout main # 切换分支

  git branch master # 创建新分支
  git checkout master

四、首次提交 | 关联 上游 分支 

  # 提交
  git add -A
  git commit -m "Commit public and set-up-blog."

  git push --set-upstream cute master  # 上游 分支

五、删除commit记录 

  git log # 最新提交 F E D C B，要删除F和E，记录D
  git rebase -i D # 将F E的pick改为drop
  git log # 查看当前提交 D C B
  git push cute master --force # 上游 分支

六、存在多个文件添加/删除/修改，在上次提交的基础上再次提交 | 不建议如此操作，风险较大

  • 项目文件夹一式两份，A和A_BAK

  • 删除文件夹A内所有代码 | 不会删除.git文件夹，提交和同步记录均保留

      rm -rf A/*

  • 将A_BAK内的所有文件拷贝到A | 不会拷贝.git

      cp -r A_BAK/* A/

  • 再次提交

      git add -A
  git commit -m "Comments."
  git push
