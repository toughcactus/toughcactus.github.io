C++ > 动态内存 > shared_ptr

一、shared_ptr

  • 多个shared_ptr共同管理一个动态对象，当销毁最后一个shared_ptr时，销毁该动态对象
  • 使用计数器记录联合管理的shared_ptr个数

要注意shared_ptr对象的生存期 

  void process(shared_ptr<int> ptr) {}
  int *x(new int(1024));
  process(shared_ptr<int>(x)); // 动态对象已被销毁
  int j = *x; // 错误

二、使用make_shared申请动态内存

  • make_shared的参数匹配对象类型的构造函数
  auto sp = make_shared<T>();
  auto sp = make_shared<T>(args);
  shared_ptr<int> p1 = make_shared<int>();
  shared_ptr<int> p2 = make_shared<int>(42);
  auto p3 = make_shared<vector<string>>();
  shared_ptr<string> p4 = make_shared<string>(10, '9');

  cout << *p1 << endl;
  cout << *p2 << endl;
  cout << *p4 << endl;

三、多个shared_ptr管理同一个动态对象

1. 使用一个shared_ptr对象初始化另一个shared_ptr对象

  • 使用sp1初始化sp2

    • sp1指向的动态对象A的引用计数为x
    • 因为动态对象A的引用计数增加，计数器的值变作x + 1
      shared_ptr<T> sp1 = make_shared<T>(args);
  shared_ptr<T> sp2(sp1);

2. 赋值运算

  • 增加管理动态对象的shared_ptr

    • sp2指向动态对象B

      • 如果B存在，其计数器加1

    • sp1指向动态对象A

      • 如果A的计数器大于1，计数器减1
      • 如果A的计数器等于1，计数器变为0，对A调用析构函数
      sp1 = sp2;

四、use_count操作 | 获取引用计数

  • 返回指向动态对象的shared_ptr个数
  sp.use_count();

五、unique操作 | 判断是否有其他shared_ptr管理该动态对象

  • 返回类型为bool
  • 只有一个shared_ptr返回true，否则返回false
  p.unique();

七、从内置指针接管动态对象，并给出可调用对象作为删除器 

  shared_ptr<T> sp(p, d);

shared_ptr指定删除器

shared_ptr指定删除器.cpp | 下载

  • 销毁动态对象时，对内置指针调用删除器，由删除器负责销毁动态对象
  • 删除器的参数是内置指针类型
  • 销毁shared_ptr对象时，会销毁其管理的动态对象

八、善用shared_ptr机制 ｜ 使用删除器管理非动态对象

  • shared_ptr指向非动态对象
  • 销毁shared_ptr对象时，对管理对象调用删除器

使用shared_ptr管理connection，销毁shared_ptr时，对connection调用end_connection

  • connection仍存在，可再次使用
  void end_connection(connection *p) { disconnect(*p); }

  void f(destination &d)
  {
      connection c = connect(&d);
      shared_ptr<connection> p(&c, end_connection); // 销毁p时会对c调用end_connection
  }

九、reset操作 | 从内置指针接管动态对象 

  sp.reset(p, d);

shared_ptr的reset操作

shared_ptr的reset操作.cpp | 下载

  • 不用显式转换

  • 第一个参数的缺省值为nullptr，第二个参数的缺省值为delete

    • 无参数 | 将sp置空
    • 给出内置指针
    • 给出内置指针和删除器 | 销毁动态对象时，对其调用删除器

十、使用shared_ptr管理动态数组

  • 使用删除器销毁动态数组

      shared_ptr<int> sp(new int[10], [](int *p) { delete[] p; });
  sp.reset();
