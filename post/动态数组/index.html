C++ > 动态内存 > 动态数组

还可以使用new和delete创建动态数组。

使用new创建动态数组时，返回的是指向首元素的指针。

一、使用new[]创建动态数组

  • new[]返回指向首元素的指针，该指针不是数组指针

    • 不能对其执行begin和end操作
    • 不能对其使用范围for
默认初始化 new + 类型 + [ + n + ]
直接初始化 new + 类型 + [ + n + ] + ( + )
列表初始化 new + 类型 + [ + n + ] + { + 初始化列表 + } 
  T *pa = new T[n];
  T *pa = new T[n]();
  T *pa = new T[n]{v1, v2...};
  int *pia = new int[10];
  string *psa = new string[10];

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << pia[i] << endl;

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << psa[i] << endl;
  int *pia = new int[10]();
  string *psa = new string[10]();

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << pia[i] << endl;

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << psa[i] << endl;
  int *pia = new int[10]{1, 2, 3};
  string *psa = new string[10]{string(10, 't'), "hello"};

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << pia[i] << endl;

  for (auto i = 0; i < 10; ++i)
    cout <<"No " << i + 1 << ": " << psa[i] << endl;

遍历动态数组 

  int *p = new int[n];
  for (int *q = p; q != p + n; ++q)
    /* 处理数组 */;

数组大小可以为0，返回尾后指针 

  auto p = new T[0];
  auto p = new T[0]();
  char *cp = new char[0];

二、使用delete[]销毁动态数组

delete + [ + ] + 指向动态数组第一个元素的指针 
  T *pa = new T[n];
  delete [] pa;
  typedef int arrT[42];
  int *p = new arrT;
  delete [] p;

接受一个空指针，或者指向动态数组的第一个元素

  • 如果指针指向单个动态对象，行为未定义
  • 如果指针指向动态数组除首元素外的其他元素，行为未定义
  • 如果使用delete销毁动态数组，行为未定义
