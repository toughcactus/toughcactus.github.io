二、插件管理

  • 插件管理器 | use-package

    • 安装

        (unless (package-installed-p 'use-package) ;; 安装use-package
    (package-refresh-contents)
    (package-install 'use-package))

    • 使用

        (use-package package-name
    :ensure t
    :config () ;; 加载插件以后 配置
    :bind () ;; 键位绑定
    :init () ;; 加载插件之前 初始化工作
    :hook()
    :custom() ;; 自定义变量
    )

  • 试用插件 | try

      (use-package try
    :ensure t)
    • 重启后删除插件

  • 列表安装

      (require 'cl-lib)

  (defvar my/packages '(spacemacs-theme
                        dracula-theme
                        rebecca-theme
                        inkpot-theme
                        farmhouse-themes
                        zeno-theme
                        ) "Default packages") ;; 列出package

  (defun my/packages-installed-p ()
    (cl-loop for pkg in my/packages
             when (not (package-installed-p pkg)) do (cl-return nil) ;; 如果有pkg未安装，返回nil
             finally (cl-return t)
             )) ;; 对列表中的package进行检查

  (unless (my/packages-installed-p) ;; 如果函数调用返回nil
    (message "%s" "Refreshing package database...")
    (package-refresh-contents) ;; 重新拉取package
    (dolist (pkg my/packages)
      (when (not (package-installed-p pkg)) ;; 如果有pkg未安装
        (package-install pkg) ;; 安装
        )))

三、Mac环境

  • 在Emacs中无法执行其他安装包的命令 | 从终端运行Emacs可以
  • 作者Purcell | Emacs用户

  • 为在Emacs中执行shell命令提供支持 | exec-path-from-shell

      (use-package exec-path-from-shell
    :ensure t
    :init
    (when (memq window-system '(mac ns x))
      (exec-path-from-shell-initialize))
    )
