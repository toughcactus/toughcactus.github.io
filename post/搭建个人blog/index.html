一、准备工作

1. 使用go安装hugo

    1. 安装go
    • 官网下载

    • 环境变量

        # go
  export PATH=/usr/local/go/bin:$PATH
  export GOROOT=/usr/local/go
  export GOPATH=/Users/USERNAME/go
  export GOBIN=/Users/USERNAME/go/bin

  # go安装的软件
  export PATH=$GOBIN:$PATH

    • 生效

        vi ~/.zshrc
  source ~/.zshrc
    1. 安装hugo

    • 如若遇上网络不佳

        go env -w GOPROXY=https://goproxy.cn,direct
  go env -w GOSUMDB=off

    • 安装 | 验证

        go install github.com/gohugoio/hugo@latest
  hugo env -v
    • 如果需要使用hugo扩展功能 | 替换hugo文件

2. 使用homebrew安装hugo 

  brew install hugo

二、使用hugo构建静态网页

  1. 初始化项目

      hugo new site /site/path

  2. 项目结构

    • config.toml | 配置文件 | config.yaml也行
    • content | 页面 | 支持md和org
    • themes | 主题
    • public | 生成的静态页面
    • archetypes | 文章模板

  3. 下载主题 anubis | anubis on hugo | anubis on github

      # 如果使用git管理静态网页项目
  git init
  git submodule add https://github.com/mitrichius/hugo-theme-anubis.git themes/anubis

  4. 参考themes/anubis/exampleSite

      cp themes/anubis/exampleSite/config.toml .
  cp themes/anubis/exampleSite/content/. content

    • config.toml说明

      • menu | about tags archive | url是链接 不是content的下级目录

    • content说明

      • about.md | about-subpage.md | about-subpage2.md | url | about和它的两个子页
      • archive.md
      • tags | post下文章tag
      • categories | post下文章分类
      • post | 所有文章 | tags | categories | categories和tags对其分类

  5. 文章模板

      +++
  author = "Dorothy"
  title = "article title"
  date = "2023-03-17"
  description = "article description"
  tags = [
"tag-1",
"tag-2",
  ]
  categories = [
"category",
  ]
  +++

  简单介绍

  <!--more-->

  正文

  6. 创建页面 | content/post

      hugo new post/first-blog.org

  7. 运行服务器端

    • 本地测试 http://localhost:1313/
      hugo server --theme=anubis --watch
  # hugo server --theme=anubis --buildDrafts --watch
  # hugo server

  8. 生成静态页面

    • 生成文件在public
      hugo --theme=anubis --baseUrl="https://toughcactus.github.io/"
  # hugo --theme=anubis --buildDrafts --baseUrl="https://toughcactus.github.io/"
  # hugo -D
  # hugo
  # hugo -F --cleanDestinationDir

  1. 创建repository | toughcactus.github.io

  2. 为github设置主题 | 创建_config.yml文件

     remote_theme: pages-themes/midnight@v0.2.0
  3. 将public内容提交到repository
