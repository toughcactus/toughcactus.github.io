C++ > 动态内存 > 智能指针

多个shared_ptr联合管理一个动态对象，最后一个shared_ptr销毁时，销毁动态对象。

weak_ptr作为伴随指针，和shared_ptr一起使用。

通过weak_ptr访问share_ptr管理的动态对象时，会检查指针是否有效；不会增加动态对象的引用计数，也不会减少。

unique_ptr同一时间只允许一个指针管理动态对象，不允许对对象进行拷贝。当指针被销毁时，动态对象也被销毁。

  • 头文件
  • shared_ptr和unique_ptr的共同操作

一、头文件 

  #include <memory>

二、shared_ptr和unique_ptr的共同操作

1. 创建空指针

  • 未指向任何动态对象

      shared_ptr<T> sp;
  unique_ptr<T> up;
      shared_ptr<string> sp1;
  shared_ptr<list<int>> sp2;

  unique_ptr<double> up1;
  unique_ptr<string> up2;

2. 指向新创建的对象 ｜ 使用new申请动态内存 

  shared_ptr<T> sp(new T(args));
  unique_ptr<T> up(new T(args));
  shared_ptr<T> sp = shared_ptr<T>(new T(args));
  unique_ptr<T> up = unique_ptr<T>(new T(args));
  shared_ptr<int> p1(new int(10));
  shared_ptr<int> p2 = shared_ptr<int>(new int(20));

  unique_ptr<int> p3(new int(42));
  unique_ptr<string> p4(new string("Stegosaurus"));
  unique_ptr<int> p5 = unique_ptr<int>(new int(10));

存在一次转换，内置指针转换为智能指针

  • 内置指针转换为智能指针的构造函数是explicit的

      shared_ptr<T> sp = new T(args); // 错误
  unique_ptr<T> up = new T(args); // 错误

3. 访问对象

  • 判断指针是否为空
  • 获取对象的引用 | 不要解引用一个空指针
  • 访问对象的成员
  p;
  *p;
  p->mem;
  shared_ptr<string> sp(new string);
  if (sp && sp->empty())
      *sp = "hi";
  cout << *sp << endl;

4. 从内置指针接管动态内存

  • 内置指针 -> 智能指针 是显式转换
  • 建议接管后，将内置指针置空
  auto p1 = new T(args);
  auto p2 = new T(args);
  auto p3 = new T(args);
  auto p4 = new T(args);
  auto p5 = new T(args);
  auto p6 = new T(args);

  shared_ptr<T> sp1(p1);
  unique_ptr<T> up1(p2);

  shared_ptr<T> sp2 = shared_ptr<T>(p3);
  unique_ptr<T> up2 = unique_ptr<T>(p4);

  sp1 = shared_ptr<T>(p5);
  up1 = unique_ptr<T>(p6);

  p1 = p2 = p3 = p4 = p5 = p6 = nullptr;

5. get操作 | 智能指针 -> 内置指针

  • 返回一个内置指针，指向智能指针管理的动态对象

      T *pt = p.get();

1. 通过内置指针使用动态对象期间，需要保证动态对象不被销毁 

  shared_ptr<string> sp(new string("hello"));
  string *ps = sp.get();
  cout << *ps << endl;
  sp = nullptr;  // 动态对象被销毁
  string s = *ps;  // 错误
  cout << s << endl;
  shared_ptr<string> sp(new string("hello"));
  string *p = sp.get();
  cout << *p << endl;
  {
      shared_ptr<string> sp2(p);
  } // 动态对象被销毁
  string foo = *sp; // 错误
  cout << foo << endl;

2. 只通过内置指针访问动态对象，不能对该指针执行delete操作，更不能将动态对象交接给其他智能指针

6. swap操作

  • 有非成员函数和成员函数两个版本
  swap(p1, p2);
  p1.swap(p2);
