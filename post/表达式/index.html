C++ > 表达式

表达式有两个维度，类型(type)和值类别(value category)

一、对象(object)

  • 有具体类型（读写规则），不是抽象类型或不完全类型，拥有内存中的一块区域
  • 可以有名字，可以没名字
  • 有自己的生存期

读写对象的途径

  • 对具名对象直接进行读写
  • 通过绑定对象的引用
  • 通过指向对象的指针

二、值(value)

  • 拥有具体类型，没有内存空间

三、表达式

  • 由运算符和运算对象组成，对表达式求值将得到一个结果

四、表达式类型

  • 对表达式求值，结果的类型
  • 可以是基础类型，可以是复合类型，也可以是类类型
  • 表达式类型不一定是具体类型

1. 函数调用表达式的类型是返回类型

  • 返回类型是左值引用时，值类别是左值
  • 返回类型是右值引用时，值类别是速亡值
  • 返回类型是其他类型时，值类别是纯右值

2. 变量表达式的类型是变量类型，值类别是左值

五、表达式值类别

  • 对表达式求值，结果的类别

  • 包括左值(lvalue)，速亡值(xvalue)，纯右值(prvalue)

    • 左值和速亡值合称泛左值(glvalue)
    • 纯右值和速亡值合称右值(rvalue)

六、表达式结果

  • 表达式结果可以是值，或者对应一个明确对象

    • 当表达式结果是值时，值类别是纯右值
    • 如果表达式结果对应一个明确对象，值类别是泛左值(左值或速亡值)
  #include <iostream>
  #include <utility>
  using std::cout;
  using std::endl;
  int fcn(int a) { return a + 1; }
  int &fcn2(int &a) { return ++a; }
  int &&fcn3(int &a) { return std::move(a); }
  int main()
  {
      int a = 3, b = 4;
      3 + 4; // 表达式类型是int，表达式值类别是纯右值，表达式结果是值
      a + b; 

      fcn(3); // 表达式类型是int，值类别是纯右值，表达式结果是值

      int &r = a;
      r; // 表达式类型是int &，值类别是左值，表达式结果对应一个明确对象

      int c = a + b;
      fcn2(c) = 10;
      fcn2(c); // 表达式类型是int &，值类别是左值，表达式结果对应一个明确对象
  
      int &&rr = 4 + 8;
      rr; // 表达式类型是int &&，值类别是左值，表达式结果对应一个明确对象

      fcn3(a); // 表达式类型是int &&，值类别是速亡值，表达式结果对应一个明确对象
      static_cast<int &&>(c); // 表达式类型是int &&，值类别是速亡值，表达式结果对应一个明确对象
      std::move(c); // 表达式类型是int &&，值类别是速亡值，表达式结果对应一个明确对象

      return 0;
  }

七、左值(lvalue)

  • 表达式值类别的一种
  • 左值表达式可以出现在赋值运算符的左侧或右侧
  • 可对左值表达式寻址
  • 可用左值引用绑定一个左值表达式
  • 左值表达式的求值结果对应一个明确对象

1. 变量表达式是左值表达式

2. 返回左值引用的函数调用表达式是左值表达式

3. 返回左值的运算符

  • 前置递增运算符
  • 前置递减运算符
  • 解引用运算符
  #include <iostream>
  using std::cout;
  using std::endl;
  int &fcn(int &a) { return ++a; }
  int main() {
      int a = 5, b = a;
      int c = a + b;
      int *p = &a;
      *p = 7;
      *p; // 类型为左值引用，值类别是左值，表达式结果对应一个明确对象a
      ++a = 10;
      ++a; // 类型为左值引用，值类别是左值，表达式结果对应一个明确对象a
      const int ci = 10;
      //  ci = 20; // 错误
      fcn(b); // 类型为左值引用，值类别是左值，表达式结果对应一个明确对象b
      ++fcn(b);
      fcn(a) = 100;
      int *p2 = &fcn(c);
      return 0;  
  }

4. 要求操作数是左值的运算符

  • 前置递增运算符
  • 前置递减运算符
  • 后置递增运算符
  • 后置递减运算符
  • 取地址运算符
  • 赋值运算符(左操作数)

八、右值(rvalue)

  • 表达式值类别的一种，包括纯右值和速亡值
  • 右值表达式只能出现在赋值运算符的右侧
  • 右值表达式只读，不可寻址
  • 可用右值引用绑定一个右值表达式
  • 右值表达式的求值结果可以是值，也可以对应一个明确对象

九、纯右值(prvalue)

  • 右值表达式的一种
  • 纯右值表达式的求值结果是值

1. 字面值是纯右值表达式

2. 纯右值表达式物质化得到速亡值表达式

3. 返回类型非引用的函数调用表达式是纯右值表达式

4. 返回纯右值的运算符

  • 后置递增运算符
  • 后置递减运算符
  • 取地址运算符 | 返回一个指针对象

5. 隐式转换的值类别是纯右值

6. 构造函数表达式的值类别是纯右值 

  #include <iostream>
  using std::cout;
  using std::endl;
  int fcn(int a) { return a + 1; }
  int main()
  {
      int a = 3;
      int b = fcn(a);
      //  fcn(a) = 5; // 错误
      fcn(b); // 表达式类型为int，值类别是纯右值，表达式结果是值
      int c = a + b;
      //  a + b = 6; // 错误
      int d = b++;
      //  b++ = 10; // 错误
      b++; // 表达式类型为int，值类别是纯右值，表达式结果是值
      //  int *p = &fcn(10); // 错误
      return 0;
  }

十、速亡值(xvalue) ｜ expiring value

  • 类型为右值引用的右值表达式
  • 右值表达式的一种
  • 速亡值表达式的求值结果对应一个明确对象

1. 返回类型为右值引用的函数表达式都是速亡值表达式

2. std::move和static_cast<T &&>是速亡值表达式

十一、泛左值(glvalue)

  • 表达式类别的一种，包括左值和速亡值
  • 泛左值表达式的求值结果对应一个明确对象
