一、clangd

1. MacOS

  • 自带clang和clangd
  • gcc和clang都是编译器，在MacOS上运行g++命令实际用的clang

2. 下载编译好的LLVM

  • clang+llvm-15.0.4下载地址

    • 15.0.5起不提供apple-x86_64的已编译版本

  • 添加环境变量

      # llvm
  export LLVMBIN=/Users/USERNAME/Packages/clang+llvm-15.0.4/bin
  export PATH=$LLVMBIN:$PATH

  • 生效

      vi ~/.zshrc
  source ~/.zshrc

二、ccls

  • 起源于cquery，可作为C/C++/Objective-C的语言服务器。
  • 支持代码补全，定义和引用查询，以及交叉引用等功能。
  • 基于Clangd+LLVM的C++库。
  • 国人开发。
  • ccls on Github
  • 需要编译ccls这个程序，以及在Emacs上安装ccls插件。

1. 使用cmake编译ccls，使用make install安装，程序路径/usr/local/bin/ccls

2. 安装插件 

  (use-package ccls
    :ensure t
    :hook ((c-mode c++-mode objc-mode cuda-mode) .
           (lambda () (require 'ccls) (lsp))))
